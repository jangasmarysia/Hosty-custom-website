Install CLI

Requirements

  • Windows 10, Mac, OSX, Linux
  • Node.js v12 or higher

Install

Mac (homebrew)

$ bre install hosty-cli

NPM

$ npm install hosty-cli

Yarn

$ yarn install hosty-cli
